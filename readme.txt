Написан метод, реализующий анимированное распределение потенциала в зависимости от номера шага для метода Якоби (animated_plot.py).
Добавлена часть задачи, рассчитывающая поле в конденсаторе (Poisson_1.py). Для нее реализован алгоритм, создающий начальное распределение заряда на конденсаторе (cap_distribution.py).
